
Using website-run scripts and a database are Graham's ideas.

Pin Monitor Program
  Keep a collection of registered add-ons.
    Add-ons can be registered.
    Add-ons can be deleted.
    Add-ons can be enabled.
    Add-ons can be disabled.
  Monitor pins of enabled registered add-ons.
  Alert website if statuses of monitored add-ons change.
    cURL
  Periodically check database Task table for new commands from website.

Website-run Scripts ( small, short-lived, python programs )
  Enabled specified pins on Pi.
  Disable specified pins on Pi.
